# WELLlabs DDA App

Geospatial watershed diagnosis and field annotation platform.

This project is an early-stage full-stack geospatial application built with:

- Django

- PostGIS

- SvelteKit

- Leaflet

- TailwindCSS

The application allows a user to:

- select a coordinate on a map

- query watershed boundaries from PostGIS

- return GeoJSON

- render watershed polygons on the frontend

This repository is the boilerplate/foundation for future development

---

# Tech Stack

## Backend

- Python 3.9+

- Django

- Django REST Framework

- PostGIS

- GDAL / GEOS

## Frontend

- SvelteKit

- TailwindCSS
